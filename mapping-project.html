Data Viz in p5.js

Home > Mini Projects > Data Viz in p5.js

Overview

The aim of this project was to create an interactive programme using processing or p5.js to the theme “Mapping”.

I decided to use p5.js due to it being easily implemented onto the web, and I wanted to gain some JavaScript knowledge to accompany my HTML & CSS.

Research

My initial idea was to attempt a data visualization chart. I created Miro and Pinterest boards of inspiration, knowing I couldn’t make the more complicated ones, I just wanted to learn how professionals display their information so I could use similar techniques in my programme.

The Idea

I picked a data set of 800 Pokemon, the reasons for this are;

  • I know the data already, so if there are evident mistakes, I can clearly identify them.
  • There is a large amount of data to work with.
  • It is more fun and would be more entertaining data to look at.

After picking the data set, I decided on what type of chart I wanted. I decided on a ‘dot matrix chart’ because the dots are spread out equally, I know I could use loops, making the creation of the visualization easier to start.


To display the different data I planned to have each individual circle be the Pokemon name, the circle colour would be the type of Pokemon (eg: Fire, Water, etc.), the size of the circles would relate to the Pokemon’s total power points. Finally, to create some interaction, I wanted to include the console to create randomised Pokemon teams.
